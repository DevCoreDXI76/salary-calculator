# Development Plan & Roadmap

## Phase 1: 개발 환경 구성 및 레이아웃 정의 (1일차)
- [ ] Vite로 React 프로젝트 스캐폴딩 생성
- [ ] Tailwind CSS 및 Lucide React 설치 및 빌드 환경 연결
- [ ] 글로벌 헤더(Header), 푸터(Footer), 기본 탭 인터페이스(App.jsx) 구축
- [ ] 상/하단 및 사이드바 영역에 광고 지면 플레이스홀더(`AdSlot` 컴포넌트) 배치

## Phase 2: 세금 계산 유틸리티 함수 검증 (1일차)
- [ ] `utils/taxCalculator.js` 파일 작성 및 다양한 연봉액 기준 유닛 테스트 진행
- [ ] 실제 사람인/잡코리아 연봉 계산기 결과와 오차범위 1% 이내인지 정합성 검증

## Phase 3: 연봉 계산기 UI 및 실시간 바인딩 구현 (2일차)
- [ ] `InputSection.jsx` 마크업 및 한글 숫자 변환 도우미(예: 50,000,000원 -> 5,000만 원) 탑재
- [ ] `ResultSection.jsx` 마크업 및 실시간 계산 결과 출력 바인딩
- [ ] Recharts 라이브러리를 활용해 세금 공제액 비중을 나타내는 도넛 차트(`TaxChart.jsx`) 구현
- [ ] 계산 결과 텍스트 복사 버튼 구현

## Phase 4: 수익화 강화를 위한 서브 콘텐츠 탑재 (3일차)
- [ ] **소비 시뮬레이터 (`Consumption.jsx`):** 실수령액 값을 넘겨받아 재미있는 환산치(스타벅스 잔수, 치킨 마리수, 저축 기간 등)를 귀여운 카드 UI로 출력
- [ ] **퇴직금 계산기 탭 (`SeveranceCalc.jsx`):** 입사/퇴사 일자 입력 폼 구현 및 퇴직금 연산 로직 연동

## Phase 5: 최적화 및 배포 (4일차)
- [ ] 모바일/태블릿/PC 전반에 걸친 반응형 웹 디자인 검수
- [ ] Vercel 플랫폼을 이용하여 무중단 호스팅 배포 완료
- [ ] 구글 애드센스 승인 신청을 위한 광고 코드 삽입 영역 활성화
